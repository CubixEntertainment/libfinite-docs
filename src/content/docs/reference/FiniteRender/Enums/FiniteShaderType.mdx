The `FiniteShaderType` enum describes a type of shader to be used in [`FiniteRenderShaderStageInfo `](/reference/finiterender/types/finiterendershaderstageinfo/)

```c
typedef enum FiniteShaderType FiniteShaderType;
enum FiniteShaderType {
    FINITE_SHADER_TYPE_VERTEX,
    FINITE_SHADER_TYPE_FRAGMENT
};
```

## Standard Usage

When creating a [`FiniteRenderShaderStageInfo`](/reference/finiterender/types/finiterendershaderstageinfo/) this enum is expected as a parameter.
