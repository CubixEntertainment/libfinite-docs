The `FiniteColorGroup` struct stores RGBA color values as doubles, used for rendering with cairo.

```c
typedef struct {
    double r;
    double g;
    double b;
    double a;
} FiniteColorGroup;
```

## Properties

| Type       | Description                             |
| ---------- | --------------------------------------- |
| `double r` | Red component (0.0 to 1.0).             |
| `double g` | Green component (0.0 to 1.0).           |
| `double b` | Blue component (0.0 to 1.0).            |
| `double a` | Alpha (opacity) component (0.0 to 1.0). |
