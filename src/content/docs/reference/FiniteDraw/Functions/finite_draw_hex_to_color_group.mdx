The `finite_draw_hex_to_color_group` function attempts to convert a hex value to a [`FiniteColorGroup`](/reference/finitedraw/types/finitecolorgroup/).

```c
void finite_draw_hex_to_color_group(char hex[7])
```

## Parameters

| Type          | Description                                 |
| ------------- | ------------------------------------------- |
| `char hex[7]` | A seven character hex string. (Ex. #f6ab31) |

## Code Example

```c
#include <finite/draw.h>

finite_draw_set_font(shell, "Kumbh Sans", false, true, 52);
FiniteColorGroup cubi = finite_draw_hex_to_color_group("#f6ab31");

finite_draw_set_draw_position(myShell, 30, 20);

finite_draw_set_text(myShell, "Hello there!", &cubi);
```

## Standard Usage

This function does not support alpha. For alpha coloring, use [`finite_draw_hex_to_color_group_alpha`](/reference/finitedraw/functions/finite_draw_hex_to_color_group_alpha/)

This function must have a valid [`FiniteShell`](/reference/finiteshell/finiteshell/)

This function uses Cairo which requires a valid cairo_surface_t in `FiniteShell.cairo_surface` beforehand.
