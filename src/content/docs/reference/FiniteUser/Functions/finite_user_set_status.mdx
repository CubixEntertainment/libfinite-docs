</Aside>

The `finite_user_set_status` enum attempts to set the status of a given user.

```c
enum FiniteUserStatus finite_user_set_status(char *user_id, char* token, char *dev,  enum FiniteUserStatus status)
```

## Properties

| Type                           | Description                                                         |
| ------------------------------ | ------------------------------------------------------------------- |
| `char *user_id`                | The user's uuid. This is different from their index id.             |
| `char *token`                  | The request token. You can get a valid token via the Cubix Website. |
| `char *dev`                    | The device id. You can get a valid device_id via the Cubix Website. |
| `enum FiniteUserStatus status` | The status to set the user to.                                      |

## Code Example

```c
FiniteAuthRequest req* = finite_user_request_auth(shell, "", "", NULL);
finite_user_get_by_id(req->user_id, myToken, myDev);

enum FiniteUserStatus status = finite_user_set_status(req->user_id, myToken, myDev);
```

## Specification

Any status that isn't `USER_STATUS_OFFLINE` is considered online.
