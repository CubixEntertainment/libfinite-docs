The `finite_input_textbox_create` function attempts to allow user input on a [`FiniteTextbox`](/reference/finiteinput/finitetextbox/types/finitetextbox/)

```c
FiniteTextbox *finite_input_textbox_create(FiniteShell *shell, FiniteTextboxDetails *details, FiniteTextboxRenderMode mode, void *data)
```

## Parameters

| Type                                  | Description                                                                                                         |
| ------------------------------------- | ------------------------------------------------------------------------------------------------------------------- |
| `FiniteShell *shell`                  | The `FiniteShell` the textbox belongs to.                                                                           |
| `FiniteTextboxDetails *details`       | The [`FiniteTextboxDetails`](/reference/finiteinput/finitetextbox/types/finitetextboxdetails/) for the the textbox. |
| `FiniteTextboxRenderMode redraw_mode` | What redraw mode should be used when the textbox changes.                                                           |
| `void *data`                          | Data to pass to the callback functions.                                                                             |

## Code Example

```c
FiniteKeyboard *kbd = finite_input_keyboard_init(myShell->display);

FiniteTextboxDetails boxDetails = {
    .x = 20,
    .y = 50,
    .width = 250,
    .height = 40,
    .text_font = "Hack",
    .isRounded = false,
    .box_radius = 0,
    .text_size = width * 0.014,
    .stroke_weight = 0,
    .text_align = TEXTBOX_ALIGN_LEFT,
    .background_color = myColor,
    .box_color = box_color,
    .text_color = finite_draw_hex_to_color_group("#3256fa"),
    .stroke_color = finite_draw_hex_to_color_group("#000000")
};

FiniteTextbox *textbox = finite_input_textbox_create(myShell, &boxDetails, TEXTBOX_RENDER_MODE_IMPLICIT, NULL);
if (!textbox) {
    FINITE_LOG_ERROR("Unable to create textbox");
}

finite_input_textbox_request_input(textbox, kbd);
```

## Specification

`FiniteTextbox`s should **not** be created manually. Developers should let [`finite_input_textbox_create`](/reference/finiteinput/finitetextbox/functions/finite_input_textbox_create) handle creating the FiniteTextbox.
