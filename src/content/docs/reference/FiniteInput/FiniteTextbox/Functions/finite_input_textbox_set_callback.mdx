<Aside>finite_input_textbox_set_callbacks was added in **v0.7.2**</Aside>

The `finite_input_textbox_set_callbacks` function attempts to define the callback functions for a [`FiniteTextbox`](/reference/finiteinput/finitetextbox/types/finitetextbox/)

```c
void finite_input_textbox_set_callback(FiniteTextbox *textbox, void (*on_focused)(FiniteTextbox *self, FiniteTextboxCommitType type, void *data), void (*on_commit) (FiniteTextbox *self, FiniteTextboxCommitType type, void *data), void (*on_done)   (FiniteTextbox *self, FiniteTextboxCommitType type, void *data))
```

## Properties

| Type                                                                                | Description                                   |
| ----------------------------------------------------------------------------------- | --------------------------------------------- |
| `FiniteTextbox *textbox`                                                            | The `FiniteTextbox` to set the callbacks for. |
| `void (*on_focused)(FiniteTextbox *self, FiniteTextboxCommitType type, void *data)` | Focused event callback.                       |
| `void (*on_commit)(FiniteTextbox *self, FiniteTextboxCommitType type, void *data)`  | Commit event callback.                        |
| `void (*on_done*)(FiniteTextbox *self, FiniteTextboxCommitType type, void *data)`   | Textbox unfocus                               |

## Code Example

```c
FiniteKeyboard *kbd = finite_input_keyboard_init(myShell->display);

FiniteTextboxDetails boxDetails = {
    .x = 20,
    .y = 50,
    .width = 250,
    .height = 40,
    .text_font = "Hack",
    .isRounded = false,
    .box_radius = 0,
    .text_size = width * 0.014,
    .stroke_weight = 0,
    .text_align = TEXTBOX_ALIGN_LEFT,
    .background_color = myColor,
    .box_color = box_color,
    .text_color = finite_draw_hex_to_color_group("#3256fa"),
    .stroke_color = finite_draw_hex_to_color_group("#000000")
};

FiniteTextbox *textbox = finite_input_textbox_create(myShell, &boxDetails, TEXTBOX_RENDER_MODE_IMPLICIT, NULL);
if (!textbox) {
    FINITE_LOG_ERROR("Unable to create textbox");
}

finite_input_textbox_set_callback(textbox, myFocusFunction, NULL, NULL);
finite_input_textbox_request_input(textbox, kbd);
```

## Specification

This function is primarily designed to be used with `TEXTBOX_RENDER_MODE_EXPLICIT` to allow developers to control what happens on specific events.
