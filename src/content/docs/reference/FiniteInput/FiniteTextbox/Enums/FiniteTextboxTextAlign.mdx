<Aside> FiniteTextboxTextAlign was added in **v0.7.2** </Aside>

A `FiniteTextboxTextAlign` refers to specific text alignment.

```c
enum FiniteTextboxTextAlign {
    TEXTBOX_ALIGN_LEFT,
    TEXTBOX_ALIGN_CENTER
};
```

## Properties

| Name                 | Value                    |
| -------------------- | ------------------------ |
| TEXTBOX_ALIGN_LEFT   | Align text to the left   |
| TEXTBOX_ALIGN_CENTER | Align text to the center |
